## Deployment Instructions (GitHub Pages)
To deploy this website for free on GitHub Pages:
1. Create a new repository on GitHub (e.g., `dr-nehru-website`).
2. Upload all the files from this folder to the repository.
3. Go to the repository **Settings** > **Pages**.
4. Under **Build and deployment**, select the `main` branch and `/ (root)` folder.
5. Click **Save**. Your website will be live at `https://your-username.github.io/dr-nehru-website/`.
